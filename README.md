# ExerciceBDD_MCD / BDD

Ce projet contient une collection de scripts SQL pour divers exercices et projets liés à la modélisation conceptuelle de données (MCD) et à la gestion de bases de données relationnelles.

## Structure du projet

- **aembouazza_employe.sql** : Script de gestion d'employés.
- **BDD_projetMVC.sql** : Script pour un projet basé sur le modèle MVC.
- **bookbiz_create.sql** : Script de création pour la base de données BookBiz.
- **Cas Enseignement.sql** : Cas d'usage pour l'enseignement.
- **combattant_BDD.sql** : Script pour la gestion de combattants.
- **entrainement.sql / entrainement2.sql** : Scripts pour la gestion d'entraînements.
- **Formation-structure.sql** : Structure de la base de données Formation.
- **gestion de notes 2.sql** : Gestion des notes (version 2).
- **gourmand.sql** : Script pour la gestion d'un restaurant ou d'une activité gourmande.
- **LoueAuto.sql** : Script pour la gestion de location de voitures.
- **Notes1.sql / Notes2.sql** : Scripts pour la gestion des notes.
- **Representation_musicale.sql / Representation_musicale_numero2.sql** : Scripts pour la gestion de représentations musicales.
- **Restaurant.sql** : Script pour la gestion d'un restaurant.

## Utilisation

1. Ouvrez le script SQL de votre choix avec votre SGBD préféré (MySQL, PostgreSQL, SQL Server, etc.).
2. Exécutez les requêtes pour créer et manipuler les bases de données.
3. Adaptez les scripts selon vos besoins pédagogiques ou professionnels.
